# 運動項目歷史紀錄快速帶入

## 概述

選擇運動項目後，在項目選擇區下方展開顯示該項目近三次的紀錄。點選任一筆可將重量和次數帶入作為初始值，加速輸入流程。

## 顯示規則

- **觸發時機**：選擇運動項目後自動顯示
- **資料來源**：該項目近 3 筆紀錄
- **排序**：按日期新到舊
- **無紀錄時**：不顯示此區塊

## 顯示格式

| 類型     | 格式範例                              |
| -------- | ------------------------------------- |
| 一般     | `40kg × 10下 × 4組` + 難易度圓點      |
| 自體重量 | `10下 × 4組` + 難易度圓點（省略重量） |

## 點選行為

使用者點選一筆歷史紀錄後：

| 欄位   | 行為                                   |
| ------ | -------------------------------------- |
| 重量   | 帶入（若為自體重量則開啟自體重量開關） |
| 次數   | 帶入                                   |
| 組數   | 歸零，讓使用者重新計數                 |
| 難易度 | 不帶入，保持預設值                     |

### UI 細節

- 標題「最近紀錄」搭配提示文字「點選帶入」
- 每筆紀錄為可點擊的卡片/列表項目
- 難易度以小圓點顯示在右側
- 整體風格與現有 UI 一致（白底圓角卡片）

### 互動細節

- 點選後該筆紀錄短暫 highlight 表示已選擇
- 帶入值後，歷史紀錄區塊保持顯示（方便切換不同設定）
- 使用者仍可手動修改任何帶入的值
